# Mini Portal de Colombia con API

##  Descripción
Este proyecto es un mini portal web que consume datos desde la API pública de Colombia (https://api-colombia.com/) y los muestra dinámicamente en el DOM utilizando JavaScript.

Se desarrollaron dos secciones:
1. **Explorador de Departamentos de Colombia**
2. **Guia turistica de Colombia**
3. **Sabores tipicos de Colombia**

Ambas páginas están conectadas mediante hipervínculos y cumplen con los requisitos solicitados.

---

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript (ES6)
- API pública de Colombia
